Trilha Desenvolvimento

Segundo uma pesquisa realizada pela McKinsey, 17% dos projetos de TI estão indo mal a ponto de colocarem em risco a existência das próprias empresas. E a razão disso vem, muitas vezes, da falta de informação, tanto dos gestores como também, em alguns casos, dos próprios desenvolvedores. Um Desenvolvimento com qualidade é a atividade de criar, testar e manter softwares. Diante disso, é importante entender a complexidade dessa atividade, principalmente se for considerado que, para desenvolver uma solução, é necessário não apenas conhecimento técnico de programação, mas também saber conduzir uma solução do ponto de vista de negócios, usabilidade, experiência, funções, entre outras questões.

HTML e CSS

O HTML é a linguagem base para se desenvolver qualquer site. É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.
O CSS é uma linguagem de folha de estilo composta por “camadas”, criado com o propósito de estilizar as páginas HTML, ou seja, definir a aparência das páginas, para deixá-las visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais.
O HTML e o CSS são utilizados para criar páginas web e atuam de forma complementar.

Orientação a Objetos

Java não é apenas uma linguagem de programação mas sim uma plataforma de desenvolvimento onde você pode construir sistemas Desktop, Web e Mobile. Neste curso você conhecerá os princípios básicos da Orientação a Objetos em Java e estará pronto para desenvolver qualquer sistema na plataforma Java. Android

Quando um dispositivo vai além e começa a facilitar vidas, o Android está por trás disso. É por isso que seu GPS evita o trânsito, seu relógio pode enviar mensagens de texto e o Assistente pode responder a perguntas. É o sistema operacional de 2,5 bilhões de dispositivos ativos. Todos eles têm a tecnologia Android: de smartphones 5G a tablets incríveis. Os apps para Android são desenvolvidos usando as diretrizes do Material Design. Essas diretrizes abordam tudo o que você precisa saber sobre o desenvolvimento do seu app, desde fluxo da experiência do usuário até design visual, movimento, fontes e muito mais.
