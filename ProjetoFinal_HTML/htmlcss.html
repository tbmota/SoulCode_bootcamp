HTML e CSS

Descrição:

A Web é a plataforma de programação mais importante do mundo, e é praticamente onipresente. Suas tecnologias principais, o HTML e o CSS, são fundamentais para a criação de qualquer site. Quem domina essas tecnologias, suas boas práticas e seus recursos mais avançados, tem espaço no mercado de programação para front-end e pode participar da criação de plataformas incríveis.

O que vamos desenvolver durante o curso:

  • criar interfaces Web com experiência rica do usuário, estruturação correta e otimizações..
  • Domínio de HTML.
  • Tudo com tableless, código semântico, outras boas práticas e técnicas avançadas.
  • Estrutura do HMTL.
  • Listas e subitens.
  • Conhecendo DIV.
  • Atributos e links.
  • iFrame e Tabelas.
  • CSS e Seletores.
  • Padding e DIVs.
  • Propriedade Position, Float e outras.
  • Menu com listas e Submenu.
  • Layouts, Overflow e outros recursos.
  • Formatação de campos em formulários.
  • Formatação de botões.

Requisitos:

  • Conhecimento em Lógica de Programação.
  • conhecimento básico de desenvolvimento Web (como funciona a Internet e o navegador).
  • O básico de informática.
  • Acesso ao Computador.
